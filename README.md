# AI Digital Transformation Dashboard

This project analyzes the impact of artificial intelligence on digital transformation using data analytics and visualization techniques.

## Project Overview
The dashboard presents key performance indicators (KPIs) related to AI investment, adoption rates, and productivity improvement to support data-driven decision-making.

## Tools & Technologies
- Power BI
- Excel
- Data Analytics

## Repository Contents
- Power BI dashboard file (zipped)
- Dataset files used for analysis
- Project documentation

## Key Insights
- Increasing AI investment over time
- Positive correlation between AI adoption and productivity
- Industry-level comparison of AI impact
